# Try it yourself

## 1. Clone and Install
```bash
# Clone the repo
git clone https://github.com/gaykov/minimal-quotes.git

# Install dependencies
yarn install
```

## 2. Run it on iOS or Android
```bash
# Run on iOS
yarn run:ios

# Run on Android
yarn run:android
```
